my_cookbook

A form created created with HTML and CSS that adds a new dish to a cookbook - in this case a google sheets doc, that works as a simple data base. The form information gets submitted and stored in the data base using an API.

API spreadsheets generates the API for your google sheets document and is implemented in the HTML file using JavaScript and AJAX.

Very easy to implement for projects that do not require a big data base and just need to store little information. Also great practice for front-end, JavaScript and AJAX :)
