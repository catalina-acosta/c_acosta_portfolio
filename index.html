Find my Midwife

This web app is an excellent work tool for midwives, providing them with a convenient way to receive new patient requests, create appointments using the calendar, and keep their work communication in one place using the chat box. My team and I developed this application using Ruby on Rails and PostgreSQL for the database. We also used AJAX and Stimulus JS to manipulate the page behavior for the chat box and implemented the Mapbox API to display a map for each appointment.

I'm proud to have been a part of this project, and I believe that this web app will be a valuable asset to the midwifery community.

my_cookbook

I created a form using HTML and CSS that allows users to add a new dish to a cookbook. The cookbook is a Google Sheets doc that works as a simple database.

When the user submits the form information, it gets stored in the database using an API implemented with JavaScript and AJAX. This solution is ideal for projects that do not require a big database and only need to store small amounts of information.
